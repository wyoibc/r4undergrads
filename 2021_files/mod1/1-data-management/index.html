3. Git and GitHub

  • Git is a program on your computer that keeps track of any changes you make to files or folders. Imagine a project folder like skunks_ that were just working on. It may contain a variety of files, such as text files, spreadsheets, figures, PDF documents etc. When Git is told to start tracking this project folder, it creates a .git hidden subdirectory where it stores all the data on incremental changes you will have made to the project over time.

  • The strength of this meticulous record keeping is that you can always revert to an earlier version of your project should you decide to discard some of the recently made changes.

  • It goes without saying that this also serves as an excellent backup system.
